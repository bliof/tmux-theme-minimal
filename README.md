# tmux-theme-minimal

## Install

### With [TPM](https://github.com/tmux-plugins/tpm)

1. Add to your `.tmux.conf`

```
set -g @plugin 'bliof/tmux-theme-minimal'
```

2. Install -> `prefix + I`

### Copy paste

Copy the theme from [theme folder](theme) to your `.tmux.config`
